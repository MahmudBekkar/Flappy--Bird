# 🐦 Flappy Bird - C# Windows Forms

This project is a simple **Flappy Bird clone** developed using **C# and Windows Forms**.

## 🎮 Game Description

The player controls a bird and tries to fly through moving obstacles without hitting them.
Each obstacle passed increases the player's score.

## 🛠 Technologies Used

* C#
* Windows Forms
* .NET

## 🎯 Features

* Bird movement with gravity
* Obstacles moving across the screen
* Collision detection
* Score counter
* Game over screen

## 🎮 Controls

| Key   | Action             |
| ----- | ------------------ |
| Space | Make the bird jump |
| Enter | Restart the game   |
